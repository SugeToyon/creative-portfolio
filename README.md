# Creative Developer Portfolio — Emir (Sugetoyon)

Welcome to my personal portfolio! This project is a 12-section, high-end web experience designed for modern creative developers. It focuses on premium typography, smooth scroll transitions, and a minimalist aesthetic.

## 🚀 Live Demo
[View Live Project](https://sugetoyon.github.io/creative-portfolio/) 

## ✨ Key Features
This portfolio is built with a "Zero-Library" mindset for core animations (mostly Vanilla CSS & JS), ensuring high performance.

- **Sticky Stacking Cards:** Projects that overlap each other during scroll.
- **Infinite Diagonal Marquee:** 9th section features a seamless, multi-directional auto-sliding gallery.
- **Dynamic Typography:** Scroll-triggered text effects including blur-to-focus and color transitions.
- **Modern Pricing Tables:** Minimalist development packages with custom UI elements.
- **Interactive Slider:** A custom-built testimonial slider for client feedback.
- **Fully Responsive:** Optimized for every screen size from desktop to mobile.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid)
- **Interactions:** Vanilla JavaScript
- **Fonts:** Playfair Display (Serif), Inter (Sans-serif)
- **Animations:** CSS Keyframes & Scroll-driven interactions
